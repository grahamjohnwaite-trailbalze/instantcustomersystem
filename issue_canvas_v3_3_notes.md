# ICS Issue Canvas v3.3

## What changed
- Replaced the checklist-style Issue Planner with a three-column drag-and-drop Issue Canvas.
- Left: produced articles, reusable component types, and partner presence block.
- Centre: one mixed running order for articles, polls, quizzes, Q&A, partner blocks, activations and other components.
- Right: block-specific inspector fields for purpose, partner, category, commitment/exclusivity, CTA, button, URL, local proof/source, content/brief and status.
- Drag from the library into the canvas.
- Drag existing canvas cards to reorder.
- Click a card to edit it.
- "Create starter copy" provides a safe structured draft/brief without inventing local facts.
- Build Pack now exports the complete mixed running order and readiness summary.

## Important
This version stores canvas state in browser localStorage per issue, preserving the current Phase 1 backend. Partner records are still entered manually until the Active Sponsors/Airtable connection is added.
